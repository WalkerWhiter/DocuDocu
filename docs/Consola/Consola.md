### ¿Qué es?
La consola del navegador permite:
- Ver errores, logs y advertencias.
- Ejecutar código JavaScript en vivo.
- Inspeccionar y modificar el DOM.
<img src="/img/console2.png" />

### ¿Para qué se usa?
- Probar código rápido sin editar archivos.
- Depurar errores.
- Manipular estilos, variables y elementos.
